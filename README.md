# Imersao Dados Python II (Alura)

Projeto da Imersao de Dados com Python II da Alura. O objetivo e explorar, limpar e analisar dados com Python, consolidando praticas de analise e visualizacao.

## Sobre o projeto

- Análise exploratoria e preparacao dos dados
- Limpeza e padronizacao de colunas
- Geração de insights com visualizacoes
- Experimentos no notebook principal

## Conteudos do repo

- `imersao_completa.ipynb`: notebook com o fluxo completo da imersao
- `app.py`: script com partes do processamento/analise
- `dados_limpos.csv` e `dados-imersao-final.csv`: bases tratadas
- `palavras.txt`: lista auxiliar usada em etapas do projeto
- `requirements.txt`: dependencias

## Como executar

1. Crie e ative um ambiente virtual
2. Instale as dependencias
3. Rode o notebook

Exemplo (PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
jupyter notebook imersao_completa.ipynb
```
